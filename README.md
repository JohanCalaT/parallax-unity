- Target: Windows, macOS, Linux

## Description

Parallax is a Unity-based technological demonstrator 
that gamifies convergence push-up therapy exercises.
The Tobii Pro Spark eye tracker acts simultaneously 
as input device and clinical measurement instrument,
capturing vergence behavior at 60Hz to adapt 
difficulty in real time and export structured 
research data per session.
